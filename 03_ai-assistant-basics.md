# 03. AI Assistant 기본 사용법

## 3.1 Assistant 시작하기

### 열기

```
Window > AI > Assistant
```

단축키: 기본 할당 없음 (직접 설정 가능)

### 인터페이스

```

│                                     │
│  [Plan Mode] [New Chat] [History]   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │  AI 응답 영역               │   │
│  │                             │   │
│  │  - 코드 생성 결과           │   │
```

## 3.2 기본 사용 예제

### 예제 1: C# 스크립트 생성

```
요청: "플레이어 이동 스크립트를 만들어줘. WASD로 이동하고, Shift로 달리기"
```

AI Assistant가:
1. 현재 씬 컨텍스트 확인
2. C# 스크립트 파일 생성
3. 요청된 기능 구현
4. 결과 표시 및 파일 저장

### 예제 2: 씬 편집

```
요청: "씬에 빨간 구를 (0, 5, 0) 위치에 만들고, Rigidbody를 추가해줘"
```

AI Assistant가:
1. GameObject 생성
2. 위치 설정
3. 머티리얼 생성 및 색상 할당
4. Rigidbody 컴포넌트 추가
5. 변경 내역 표시

### 예제 3: 문제 해결

```
요청: "콘솔에 NullReferenceException이 뜨는데 원인을 찾아줘"
```

AI Assistant가:
1. 콘솔 로그 분석
2. 문제 코드 위치 식별
3. 수정 제안 또는 직접 수정

## 3.3 Plan Mode

**Plan Mode**는 AI가 실행 전에 수행할 작업을 **미리 보여주는** 기능입니다.

### 사용 방법

1. Assistant 창에서 **Plan Mode** 토글 켜기
2. 작업 요청 입력
3. AI가 실행 계획을 단계별로 표시
4. 각 단계를 승인/거부/수정
5. 모든 단계 승인 후 일괄 실행

### 예시

```
사용자: "HP 바 UI를 만들어줘"

[Plan Mode ON]
AI 계획:
  ✅ 1. Canvas 생성 (Scene: Main)
  ✅ 2. Slider (HealthBar) 추가
  ✅ 3. Fill Area 색상 빨강→초록 그라디언트
  ✅ 4. HealthBarController.cs 스크립트 생성
  ✅ 5. 스크립트를 Slider에 연결
  
[Accept All] [Reject] [Modify]
```

## 3.4 Skills 사용하기

### 내장 Skills

| Skill | 설명 |
|-------|------|
| Create Script | C# 스크립트 생성 |
| Modify Scene | 씬 내 GameObject 편집 |
| Analyze Project | 프로젝트 구조 분석 |
| Generate Material | 머티리얼 생성 |
| Debug | 버그 분석 및 수정 |

### Skill 실행

```
요청: "Generate Material skill로 대리석 느낌의 머티리얼을 만들어줘"
```

## 3.5 Rollback (변경 되돌리기)

AI Assistant의 모든 변경 사항은 **Rollback** 가능합니다.

- Assistant 창의 **변경 내역(History)** 탭에서 확인
- 각 변경 단계별로 **Undo** 버튼 제공
- 이전 상태로 즉시 복구

## 3.6 유용한 프롬프트 예시

### 스크립트 관련

```
"주인공이 점프할 때 2단 점프가 가능한 스크립트를 작성해줘"
"인벤토리 시스템을 ScriptableObject 기반으로 만들어줘"
"씬에 있는 모든 적의 체력을 절반으로 줄이는 에디터 스크립트를 만들어줘"
```

### 씬 편집 관련

```
"모든 라이트의 그림자 설정을 ShadowMask로 변경해줘"
"Tag가 'Enemy'인 모든 오브젝트를 찾아서 리스트로 보여줘"
"메인 카메라를 씬 상단에서 내려다보는 탑뷰로 설정해줘"
```

### 분석 관련

```
"이 씬의 퍼포먼스 병목을 분석해줘"
"현재 프로젝트에 사용된 모든 셰이더를 알려줘"
"프리팹 중에서 가장 많이 사용된 컴포넌트 TOP 5를 알려줘"
```

## 3.7 주의사항

- **구체적으로 요청할수록** 더 나은 결과를 얻을 수 있음
- Plan Mode 사용 시 예상치 못한 변경을 방지
- 중요한 작업 전에는 **프로젝트 백업** 권장
- 크레딧 소모량은 작업 복잡도에 따라 다름
